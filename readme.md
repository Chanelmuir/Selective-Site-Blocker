# Selective Site Blocker

Selective site blocker is a simple extension that lets you block sites, with specificity. I created it for the goal of blocking social media sites, but retaining their messaging functionality.

## Installation

Use the chrome extension manager. 
Click load unpacked extensions and select this folder

```bash
chrome://extensions
```

I was able to install this extension on mobile by enabling developer tools in the Microsoft Edge Canary Browser. You can then load the .crx file found in this github, or pack the folder into a .crx yourself using the chrome browser on desktop.
